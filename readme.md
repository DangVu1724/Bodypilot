# BodyPilot

## 📌 Overview

BodyPilot is a mobile application that helps users build personalized nutrition plans and workout routines based on their physical information and goals.

## 🏗️ Tech Stack

* **Frontend:** Flutter
* **Backend:** Spring Boot
* **Database:** PostgreSQL

## 🚀 Features

* Calculate BMR and TDEE
* Personalized meal recommendations
* Workout plan generation
* Track daily nutrition and exercise progress
* Chatbot support for fitness and nutrition

## 📁 Project Structure

```
fitness-app/
├── mobile/        # Flutter app
├── backend/       # Spring Boot API
├── docs/          # Documentation
├── database/      # SQL scripts
```

## ⚙️ Setup

### Backend

```bash
cd backend
./mvnw spring-boot:run
```

### Mobile

```bash
cd mobile
flutter pub get
flutter run
```
